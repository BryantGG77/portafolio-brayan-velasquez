Algunos de mis proyectos mas destacados son los siguientes:

Landing page funcional que replica una web de viajes. Esta página cuenta con aproximadamente seis secciones, entre las que se incluyen la compra de viajes específicos, tours, experiencias de usuarios y más. La información de estas secciones se gestiona mediante un backend con una base de datos encargada de almacenar los datos de viajes y tours. Además, implementé un sistema de login cuya información también se guarda en la base de datos. La página incluye una sección de administración de perfiles para verificar y eliminar usuarios, demostrando así el manejo completo de usuarios.

Este proyecto fue realizado principalmente con React para el Front, Fast API para el Back y PosgreSQL para la base de datos.

Acceder al proyecto en GitHub.

Este proyecto permite calcular la Tasa Metabólica Basal (TMB) del usuario. La aplicación toma en cuenta factores como la edad, peso, altura, nivel de actividad física semanal y género para ofrecer un cálculo preciso y personalizado.

Este proyecto fue realizado en HTML, CSS y JavaScript.

"Adivina el número secreto" es un juego interactivo que genera un número aleatorio entre 1 y 10. El objetivo del juego es adivinar el número correcto. A medida que ingresas tus respuestas, el juego te indicará si el número secreto es mayor o menor que tu suposición. Cuando finalmente aciertes, el juego mostrará la cantidad de intentos que necesitaste para lograrlo y habilitará la opción para reiniciar el juego y comenzar de nuevo.

Este proyecto fue realizado en HTML, CSS y JavaScript.
